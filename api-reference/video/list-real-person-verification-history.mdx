---
title: "List Real-Person Verification History"
openapi: "GET /v1/videos/assets/visual-validation-sessions"
description: "Lists real-person material verification sessions for the current organization."
---

List previous verification sessions for review, customer support, and reconciliation. This history is long-lived, but it does not replace the short-lived bind step: a successful verification should still be bound immediately within the 120-second window.

The response includes only AI Sonar IDs and does not include credential data. After a bind succeeds, `group_id` is the AI Sonar material group ID used for `liveness_face` uploads.

## Example Request

```bash
curl "https://api.aisonar.dev/v1/videos/assets/visual-validation-sessions?page_size=20" \
  -H "Authorization: Bearer $API_KEY"
```
