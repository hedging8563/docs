---
title: "Bind Real-Person Verification Result"
openapi: "POST /v1/videos/assets/visual-validation-results"
description: "Binds a successful real-person material verification result to a AI Sonar material group."
---

Bind the successful verification result and create the AI Sonar real-person material group mapping. Use the `session_id` and `verification_token` from the AI Sonar callback redirect.

AI Sonar verifies that the session belongs to the same organization and has not expired. The response returns a AI Sonar `group_id` only; use it when creating `liveness_face` material assets.

## Example Request

```bash
curl https://api.aisonar.dev/v1/videos/assets/visual-validation-results \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"session_id":"svv_abc123","verification_token":"svvt_def456","group_name":"Creator A"}'
```

## Example Response

```json
{
  "object": "seedance.visual_validation_result",
  "group_id": "group-20260720123456-real1",
  "material_group": {
    "id": "group-20260720123456-real1",
    "library_type": "liveness_face",
    "status": "ACTIVE",
    "group_name": "Creator A"
  }
}
```
