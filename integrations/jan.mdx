## Overview

Jan can connect to AI Sonar as a remote OpenAI-compatible provider.

<Note>
**Type**: Chat App

**Primary Path**: OpenAI-compatible chat completions

**Support Confidence**: Supported path
</Note>

## Configure Jan

1. Open **Settings** -> **Providers**
2. Add a remote OpenAI-compatible provider
3. Enter:
   - **Name**: AI Sonar
   - **Base URL**: `https://api.aisonar.dev/v1`
   - **API Key**: `sk-your-api-key`
4. Add or select `gpt-5.4-mini`
5. Save and start a new thread.

## Model Discovery

AI Sonar exposes model discovery at:

```text
https://api.aisonar.dev/v1/models
```

If your Jan build supports remote model discovery for AI Sonar, refresh the model list. Otherwise, add model ids manually.

## Recommended Models

- `gpt-5.4-mini`
- `gpt-5.4`
- `claude-sonnet-5`
- `gemini-3.5-flash`
- `deepseek-v4-flash`
