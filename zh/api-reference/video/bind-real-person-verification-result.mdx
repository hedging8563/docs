---
title: "绑定真人验证结果"
openapi: "POST /v1/videos/assets/visual-validation-results"
description: "将成功的真人材料验证结果绑定至 AI Sonar 材料组。"
---

绑定成功的验证结果并创建 AI Sonar 真人材料组映射。请使用来自 AI Sonar 回调重定向中的 `session_id` 和 `verification_token`。

AI Sonar 会验证该会话是否属于同一组织、是否未过期，以及是否绑定至验证会话所使用的同一材料服务。响应仅返回 AI Sonar 的 `group_id`；在创建 `liveness_face` 材料资产时请使用该 ID。

## 请求示例

```bash
curl https://api.aisonar.dev/v1/videos/assets/visual-validation-results \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"session_id":"svv_abc123","verification_token":"svvt_def456","group_name":"Creator A"}'
```

## 响应示例

```json
{
  "object": "seedance.visual_validation_result",
  "group_id": "seedance_group_real_123",
  "material_group": {
    "id": "seedance_group_real_123",
    "library_type": "liveness_face",
    "status": "ACTIVE",
    "group_name": "Creator A"
  }
}
```
