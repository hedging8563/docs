---
title: "删除素材资源组"
openapi: "DELETE /v1/videos/assets/groups/{groupId}"
description: "删除当前组织拥有的 Seedance 素材组。"
---

删除一个素材组及其内部的 AI Sonar 素材资源。此操作不可撤销。

如果 AI Sonar 素材库因当前授权状态限制而无法完成删除，AI Sonar 将返回一个中性的素材库错误。删除后，该组及其资源将无法再用于视频生成。

## 请求示例

```bash
curl https://api.aisonar.dev/v1/videos/assets/groups/seedance_group_virtual_123 \
  -X DELETE \
  -H "Authorization: Bearer $API_KEY"
```

## 响应示例

```json
{
  "deleted": true,
  "id": "seedance_group_virtual_123"
}
```
