---
title: "列出真人验证历史记录"
openapi: "GET /v1/videos/assets/visual-validation-sessions"
description: "列出当前组织下的真人素材验证会话。"
---

列出之前的验证会话，以便进行审查、客户支持和对账。此历史记录具有长效性，但不能替代短效的绑定步骤：成功的验证仍应在 120 秒的时间窗口内立即完成绑定。

响应仅包含 AI Sonar ID，不包含凭证数据。绑定成功后，`group_id` 即为用于 `liveness_face` 上传的 AI Sonar 素材组 ID。

## 请求示例

```bash
curl "https://api.aisonar.dev/v1/videos/assets/visual-validation-sessions?page_size=20" \
  -H "Authorization: Bearer $API_KEY"
```
