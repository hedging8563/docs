---
title: "创建真人验证会话"
openapi: "POST /v1/videos/assets/visual-validation-sessions"
description: "创建一个 AI Sonar 真人素材验证 H5 会话。"
---

创建一个短效验证会话，并将您的用户重定向到返回的 H5 链接。AI Sonar 仅返回中立字段：`session_id`、`h5_link`、`callback_url`、`expires_in` 和 `language`。

会话和 H5 链接将在 120 秒后过期。绑定 token 只会通过 AI Sonar 回调重定向提供。当 H5 流程完成后，AI Sonar 会将用户重定向回您原始的 `callback_url`，并附带以下查询参数：

| 参数 | 含义 |
|-----------|---------|
| `session_id` | AI Sonar 验证会话 ID。 |
| `verification_token` | 用于绑定结果的不透明 token。 |
| `verification_status` | `succeeded` 或 `failed`。 |
| `result_code` | 用于您应用程序逻辑的结果代码。 |

如果 `verification_status` 为 `succeeded`，请调用 [Bind Real-Person Verification Result](/zh/api-reference/video/bind-real-person-verification-result)。在将此流程接入您的产品之前，可以使用 [test console](https://console.aisonar.dev/seedance-assets) 来验证回调处理。

## 请求示例

```bash
curl https://api.aisonar.dev/v1/videos/assets/visual-validation-sessions \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"callback_url":"https://yourapp.example.com/seedance/callback","language":"en"}'
```

## 响应示例

```json
{
  "object": "seedance.visual_validation_session",
  "session_id": "svv_abc123",
  "h5_link": "https://verify.example.com/h5?lng=en",
  "callback_url": "https://yourapp.example.com/seedance/callback",
  "expires_in": 120,
  "language": "en"
}
```
