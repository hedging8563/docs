---

## 概述

删除通过 `/v1/files` 创建的文件。不带 Anthropic 请求头时，此路由删除 AI Sonar 批量文件，或已绑定的图像/参考文件绑定。带 `anthropic-beta: files-api-2025-04-14` 时，此路由按绑定工作区使用 Anthropic Files API 模式。

## 注意事项

- 删除批量文件会移除已存储的批量文件记录，并阻止之后的批量作业继续使用该文件 ID。
- 删除图像/参考文件会先请求所选提供商删除文件，提供商接受删除后再移除 AI Sonar 文件绑定。
- Anthropic Files API 模式需要使用与上传、检索相同的 Anthropic 原生请求头。

## 示例

<RequestExample>
```bash cURL
curl -X DELETE "https://api.aisonar.dev/v1/files/file_abc123" \
  -H "Authorization: Bearer sk-your-api-key"
```
</RequestExample>
