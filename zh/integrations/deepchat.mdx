## 概述

DeepChat 通过标准的 OpenAI 兼容提供商配置与 AI Sonar 协同工作。

<Note>
**类型**: 聊天应用

**主要路径**: OpenAI 兼容的聊天补全 (chat completions)

**支持置信度**: 支持的路径
</Note>

## 配置 DeepChat

1. 打开 **Settings** -> **Model Provider**
2. 添加或编辑一个 OpenAI 兼容的提供商
3. 输入以下信息：
   - **Provider name**: AI Sonar
   - **Base URL**: `https://api.aisonar.dev/v1`
   - **API Key**: `sk-your-api-key`
4. 添加一个模型，例如 `gpt-5.4-mini`
5. 保存并测试聊天功能。

## 模型选择

从 `gpt-5.4-mini` 开始，这是一个低门槛的默认选择。当对话需要更强的推理能力时，请使用 `claude-sonnet-5`、`gpt-5.4` 或 `deepseek-v4-pro`。

## 故障排除

- 请使用 base URL，而不是完整的 `/chat/completions` 路径，除非 DeepChat 明确要求输入完整的端点 URL。
- 请移除 API Key 末尾的空格。
- 如果模型未列出，请手动添加准确的 AI Sonar 模型 ID。
