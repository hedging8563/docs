## 概述

OpenLLMetry 的 OpenAI 插桩（instrumentation）功能可以追踪通过官方 OpenAI SDK 发起的调用。请使用 AI Sonar 的 `/v1` 基础 URL 配置该 SDK。

<Note>
**类型**: 可观测性

**主要路径**: OpenAI SDK 插桩

**支持置信度**: 支持兼容 OpenAI 的可观测性路径
</Note>

## 安装

```bash
pip install opentelemetry-instrumentation-openai openai
```

## 环境配置

```bash
export API_KEY="sk-your-api-key"
```

## 示例

```python
import os

from openai import OpenAI
from opentelemetry.instrumentation.openai import OpenAIInstrumentor

OpenAIInstrumentor().instrument()

client = OpenAI(
    api_key=os.environ["API_KEY"],
    base_url="https://api.aisonar.dev/v1",
)

client.chat.completions.create(
    model="claude-sonnet-5",
    messages=[{"role": "user", "content": "Trace this request"}],
)
```

## 端点说明

请将此路径用于兼容 OpenAI SDK 的 AI Sonar 调用。对于 AI Sonar 原生端点，请对实际发送这些请求的 HTTP 或 SDK 客户端进行插桩。
