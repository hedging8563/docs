## 概述

当 OpenAI SDK 配置为使用 AI Sonar 的 `/v1` 基础 URL 时，Langfuse 可以通过其 OpenAI 集成来追踪 AI Sonar 的调用。

<Note>
**类型**: 可观测性 (Observability)

**主要路径**: OpenAI SDK 追踪

**支持置信度**: 支持的 OpenAI 兼容追踪路径
</Note>

## 环境配置

```bash
export LANGFUSE_SECRET_KEY="sk-lf-..."
export LANGFUSE_PUBLIC_KEY="pk-lf-..."
export LANGFUSE_BASE_URL="https://cloud.langfuse.com"
export API_KEY="sk-your-api-key"
```

## 示例

```python
import os

from langfuse import observe
from langfuse.openai import openai

openai.api_key = os.environ["API_KEY"]
openai.base_url = "https://api.aisonar.dev/v1"

@observe()
def story():
    return openai.chat.completions.create(
        model="claude-sonnet-5",
        messages=[{"role": "user", "content": "Write a short haiku."}],
    ).choices[0].message.content
```

## 端点说明

本页面涵盖了针对 OpenAI SDK 路径的 Langfuse 追踪。如果您的应用程序使用 AI Sonar 原生响应 (Responses)、Anthropic Messages 或 Gemini 端点，请确保您的插桩 (instrumentation) 能够捕获该客户端调用路径。
