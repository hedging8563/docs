## 概述

Graphiti 通过 `OpenAIGenericClient` 支持兼容 OpenAI 的 `/v1` 端点。将该客户端指向 AI Sonar 并选择一个 AI Sonar 模型即可。

<Note>
**类型**: 知识图谱记忆框架

**主要路径**: 兼容 OpenAI 的 Chat Completions

**支持置信度**: 支持兼容 OpenAI 的路径
</Note>

## 环境

```bash
export API_KEY="sk-your-api-key"
```

## LLM 客户端

```python
import os

from graphiti_core.llm_client.config import LLMConfig
from graphiti_core.llm_client.openai_generic_client import OpenAIGenericClient

llm_config = LLMConfig(
    api_key=os.environ["API_KEY"],
    model="claude-sonnet-5",
    small_model="gpt-5.4-mini",
    base_url="https://api.aisonar.dev/v1",
)

llm_client = OpenAIGenericClient(config=llm_config)
```

将 `llm_client` 与您的图数据库和嵌入器配置一起传入 `Graphiti(...)` 实例中。

## 端点说明

Graphiti 的兼容 OpenAI 路径是面向 chat-completions 的。请使用能够可靠返回结构化 JSON 以进行提取和去重任务的模型。
