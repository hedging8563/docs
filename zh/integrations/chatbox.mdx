## 概述

Chatbox 可以通过其 OpenAI 兼容的 API 设置使用 AI Sonar。

<Note>
**类型**: 聊天应用

**主要路径**: OpenAI 兼容的聊天补全 (chat completions)

**支持置信度**: 支持的路径
</Note>

## 配置 Chatbox

1. 打开 **Settings** -> **AI Model**
2. 选择 **OpenAI API Compatible**
3. 输入以下内容：
   - **API Host**: `https://api.aisonar.dev/v1`
   - **API Key**: `sk-your-api-key`
   - **Model**: `gpt-5.4-mini`
4. 保存并发送一条简短的测试消息。

## 推荐模型

| 使用场景 | 模型 |
|----------|-------|
| 日常聊天 | `gpt-5.4-mini` |
| 更强的推理能力 | `gpt-5.4` 或 `gpt-5.5` |
| Claude 风格输出 | `claude-sonnet-5` |
| 快速回退 | `deepseek-v4-flash` |
