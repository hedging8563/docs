## 概述

Opik 可以通过封装指向 AI Sonar OpenAI 兼容端点的 OpenAI 客户端来记录 AI Sonar 的调用。

<Note>
**类型**: 可观测性 (Observability)

**主要路径**: OpenAI SDK 追踪封装器

**支持置信度**: 支持的 OpenAI 兼容追踪路径
</Note>

## 安装

```bash
pip install opik openai
```

## 环境配置

```bash
export API_KEY="sk-your-api-key"
export OPIK_PROJECT_NAME="aisonar-integration-demo"
```

## 封装 OpenAI 客户端

```python
import os

from openai import OpenAI
from opik.integrations.openai import track_openai

client = OpenAI(
    api_key=os.environ["API_KEY"],
    base_url="https://api.aisonar.dev/v1",
)

openai_client = track_openai(client)
```

## 记录聊天补全 (Chat Completion)

```python
completion = openai_client.chat.completions.create(
    model="claude-sonnet-5",
    messages=[
        {"role": "user", "content": "Summarize why LLM observability matters."}
    ],
)

print(completion.choices[0].message.content)
```

## 端点说明

此设置用于追踪 OpenAI 兼容的 SDK 路径。如果您在 OpenAI SDK 之外使用 AI Sonar 原生端点，请单独对该客户端或工作流进行插桩 (instrument)。
