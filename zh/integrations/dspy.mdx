## 概述

DSPy 可以通过使用 `openai/` 提供程序前缀以及 AI Sonar 的 OpenAI 兼容 `/v1` 端点来调用 AI Sonar。

<Note>
**类型**: 编程框架

**主要路径**: OpenAI 兼容语言模型调用

**支持置信度**: 支持自定义端点路径
</Note>

## 环境

```bash
export API_KEY="sk-your-api-key"
```

## 配置 DSPy

```python
import os

import dspy

lm = dspy.LM(
    "openai/claude-sonnet-5",
    api_key=os.environ["API_KEY"],
    api_base="https://api.aisonar.dev/v1",
)

dspy.configure(lm=lm)
```

## 使用模块

```python
qa = dspy.ChainOfThought("question -> answer")

response = qa(question="What should I monitor in an LLM gateway?")
print(response.answer)
```

## 模型建议

使用 `gpt-5.4-mini` 进行快速实验，使用 `claude-sonnet-5` 获取更强的推理轨迹，并使用 `deepseek-v4-flash` 进行快速迭代循环。
