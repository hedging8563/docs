## 概述

LibreChat 可以通过自定义的 OpenAI 兼容端点与 AI Sonar 良好协作。

<Note>
**类型**: 聊天应用

**主要路径**: OpenAI 兼容的聊天补全 (chat completions)

**支持置信度**: 支持的路径
</Note>

## 环境

```bash
API_KEY=sk-your-api-key
```

## 端点示例

将 AI Sonar 添加到您的 LibreChat 端点配置中：

```yaml
customEndpoints:
  - name: 'AI Sonar'
    apiKey: '${API_KEY}'
    baseURL: 'https://api.aisonar.dev/v1'
    models:
      default:
        - 'gpt-5.5'
        - 'gpt-5.4'
        - 'gpt-5.4-mini'
        - 'claude-opus-4-8'
        - 'claude-sonnet-5'
        - 'gemini-3.5-flash'
        - 'deepseek-v4-pro'
        - 'deepseek-v4-flash'
      fetch: true
    titleConvo: true
    titleModel: 'gpt-5.4-mini'
    modelDisplayLabel: 'AI Sonar'
```

## 注意事项

LibreChat 在此设置中使用 OpenAI 兼容的聊天路径。对于支持相应请求格式的客户端，AI Sonar 还提供了原生的 Responses、Anthropic Messages 以及 Gemini 兼容端点。
