## 概述

Open WebUI 支持自定义兼容 OpenAI 的 API 连接。您可以将 AI Sonar 添加为远程兼容 OpenAI 的端点，并刷新模型列表。

<Note>
**类型**: Web 聊天应用

**主要路径**: 兼容 OpenAI 的聊天补全 (chat completions)

**支持置信度**: 支持的聊天应用路径
</Note>

## 管理员连接

1. 打开 **Admin Panel** -> **Settings** -> **Connections**。
2. 在 OpenAI API 连接区域，添加一个连接。
3. 将 Base URL 设置为 `https://api.aisonar.dev/v1`。
4. 将 API key 设置为您的 AI Sonar 密钥。
5. 保存连接并刷新模型。

## 数值

| 字段 | 数值 |
|-------|-------|
| API Base URL | `https://api.aisonar.dev/v1` |
| API Key | `sk-your-api-key` |
| 示例模型 | `gpt-5.4-mini` |

## 推荐模型

- `gpt-5.4-mini` 用于通用聊天
- `claude-sonnet-5` 用于写作和推理
- `deepseek-v4-pro` 用于代码密集型聊天
- `gemini-3.5-flash` 用于快速且支持多模态的工作流

如果您的部署中禁用了模型发现功能或模型已缓存，请在 Open WebUI 的模型设置中手动添加模型 ID。
