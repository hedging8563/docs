## 概述

Browser Use 可以通过 `ChatOpenAILike`（即兼容 OpenAI 的聊天模型封装器）调用 AI Sonar。

<Note>
**类型**: 浏览器自动化智能体框架

**主要路径**: 兼容 OpenAI 的 Chat Completions

**支持置信度**: 支持兼容 OpenAI 的路径
</Note>

## 环境配置

```bash
export API_KEY="sk-your-api-key"
```

## 示例

```python
import os

from browser_use import Agent
from browser_use.llm.openai.like import ChatOpenAILike

llm = ChatOpenAILike(
    model="claude-sonnet-5",
    api_key=os.environ["API_KEY"],
    base_url="https://api.aisonar.dev/v1",
)

agent = Agent(
    task="Compare the top search results for AI Sonar integrations",
    llm=llm,
)
```

## 端点说明

对于通过兼容 OpenAI 的客户端调用 Chat Completions 的 Browser Use 工作流，请使用此页面。如果您的工作流需要 AI Sonar 原生响应、Anthropic Messages 或 Gemini 端点，请在 Browser Use 中使用直接支持该协议的客户端路径。
