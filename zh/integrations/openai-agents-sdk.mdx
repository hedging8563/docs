## 概述

OpenAI Agents SDK 可以通过兼容 OpenAI 的客户端和 `OpenAIChatCompletionsModel` 来使用 AI Sonar。

<Note>
**类型**: Python Agent SDK

**主要路径**: 兼容 OpenAI 的聊天补全 (chat-completions)

**支持置信度**: 支持的聊天补全路径
</Note>

## 环境

```bash
export API_KEY="sk-your-api-key"
export API_BASE_URL="https://api.aisonar.dev/v1"
export MODEL_ID="gpt-5.4-mini"
```

## 示例

```python
import asyncio
import os

from openai import AsyncOpenAI

from agents import Agent, OpenAIChatCompletionsModel, Runner, set_tracing_disabled

client = AsyncOpenAI(
    base_url=os.getenv("API_BASE_URL", "https://api.aisonar.dev/v1"),
    api_key=os.environ["API_KEY"],
)

set_tracing_disabled(disabled=True)

async def main():
    agent = Agent(
        name="Assistant",
        instructions="You are concise and practical.",
        model=OpenAIChatCompletionsModel(
            model=os.getenv("MODEL_ID", "gpt-5.4-mini"),
            openai_client=client,
        ),
    )

    result = await Runner.run(agent, "Explain custom model providers in one paragraph.")
    print(result.final_output)

asyncio.run(main())
```

## 范围

此路径使用 Chat Completions 兼容性。AI Sonar 的原生 Responses 端点位于 `https://api.aisonar.dev/v1/responses`，但在生产环境使用前，应针对具体模型和工作流验证依赖于 OpenAI 特定 Responses 语义的 SDK 功能。
